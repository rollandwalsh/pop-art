Typed Plugin. Enter in any string, and watch it type at the speed you've set :).

== Description ==

* Typed Plugin created based on "typed.js" a jQuery plugin that types. Enter in any string, and watch it type at the speed you've set, backspace what it's typed, and begin a new sentence for however many strings you've set.

* With simple UI insert your strings and configure how you want it to be displayed, then copy/paste the generate ShortCode to you page/widgate.

* Also you can manage your old generated strings an update them.

== Installation ==

1. Download "Typed" plugin and upload it to the /wp-content/plugins/ directory.
2. Activate "Typed" plugin through the Plugins menu.
